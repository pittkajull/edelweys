# UI Redesign Plan — Claude-like Clean Aesthetic

## Problem Statement
Current UI feels messy, boring, and inconsistent. User wants Claude's clean, minimal, smooth aesthetic.

## Key Issues Identified
1. **Landing**: Excessive padding (py-24/py-32), oversized text, too spread out
2. **Chat**: Sidebar opens by default (should be collapsed), boring hamburger menu
3. **All pages**: Too many inline styles, inconsistent spacing, dead CSS utilities
4. **Typography**: DM Serif Display creates clash, should use single clean font
5. **Colors**: Too many gradients, should be more minimal

## Design Direction: Claude-inspired Clean
- **Typography**: Single font family (Plus Jakarta Sans), no serif display font
- **Colors**: Muted greens + warm neutrals, minimal gradients
- **Spacing**: Tighter, more intentional whitespace
- **Components**: Clean inputs, simple buttons, subtle shadows
- **Layout**: Sidebar collapsed by default, responsive

## Implementation Plan

### Phase 1: Foundation (tailwind.config.js + index.css)
**Files**: `tailwind.config.js`, `src/index.css`

- Remove DM Serif Display font family
- Simplify color palette to 5 core colors
- Remove dead CSS utility classes (.glass, .btn-primary, etc.)
- Add consistent focus styles
- Clean up scrollbar styling

### Phase 2: Landing Page
**File**: `src/pages/Landing.jsx`

- **Navbar**: Reduce padding to `px-6 py-3`, remove background blur, simple border-bottom
- **Hero**: 
  - Reduce padding from `py-24 md:py-36` → `py-16 md:py-24`
  - Reduce heading from `text-5xl md:text-7xl` → `text-4xl md:text-5xl`
  - Remove "AI Health Companion" badge (too corporate)
  - Simplify CTA buttons - one primary, one text link
- **Features**:
  - Reduce section padding from `py-24 md:py-32` → `py-16 md:py-20`
  - Simpler card design, less glass effect
  - Reduce section heading from `text-4xl md:text-5xl` → `text-3xl md:text-4xl`
- **CTA**: Reduce padding, simpler design
- **Footer**: Minimal, clean

### Phase 3: Chat Page
**File**: `src/pages/Chat.jsx`

- **Sidebar**: Default to closed (`useState(false)`)
- **Hamburger menu**: 
  - Replace 3-line icon with a cleaner sidebar toggle
  - Add subtle animation
- **Header**: 
  - Cleaner design, less glass effect
  - Simpler button styles
- **Messages**: 
  - Reduce gap from `gap-4` → `gap-3`
  - Cleaner message bubbles
- **Input area**: 
  - Simpler textarea, no glass effect
  - Cleaner send button

### Phase 4: Auth Pages
**Files**: `src/pages/Login.jsx`, `src/pages/Register.jsx`

- **Card**: Reduce padding from `p-10` → `p-8`
- **Logo**: Reduce from `h-20` → `h-16`
- **Title**: Use sans-serif, reduce size
- **Inputs**: Simpler focus states
- **Buttons**: Consistent style (green gradient for both)
- **Remove**: Logo rotation animation

### Phase 5: Dashboard
**File**: `src/pages/Dashboard.jsx`

- **Header**: Simpler, less decorative
- **Tabs**: Cleaner tab design
- **Cards**: Simpler GlassCard, less glass effect
- **Charts**: Cleaner chart styling
- **Forms**: Simpler input styling
